アーバンデータチャレンジとは

一般社団法人社会基盤情報流通推進協議会（AIGID）が主催する公共データを活用したアプリやサービスの一般参加型のコンテストのことです．地域課題の解決を目的としていることが特徴であり，今年度は全都道府県に地域拠点が設置されました．各地域拠点ではアイデアソンやハッカソンと呼ばれるイベントが実施され，コンテストに応募するための作品開発が行われます．愛知拠点でも複数回のイベントを予定していますので，市民の皆さんのご参加をお待ちしています．

愛知拠点の特徴

愛知拠点では愛知県内のシビックテック組織である
Code for NAGOYA，Code for MIKAWA，Code for AICHIが
タッグを組んで問題解決に取り組みます．
愛知拠点のスローガンは「市民からのありがとう❤」です．
多くの市民を活動に巻き込み，市民からたくさんの感謝の言葉を
かけてもらうことが今年度の目標です！

活動計画

愛知拠点で今後実施を予定しているイベントの一覧です．

アイデアソン for チャレンジド

開催日時：2018年10月13日（土）10:00～
会場：椙山女学園大学 椙山人間交流会館・キャリア支援ルーム　
主催：アーバンデータチャレンジ愛知拠点 事務局
共催：椙山女学園大学・障がい者支援サークルYELLTECH 
詳細：http://web.sugiyama-u.ac.jp/~nmukai/ideathon-challenged/

チャレンジド（Challenged）は「障がいを持つ人」を意味する新しい表現です． 障がいを人生の使命や課題と捉え，挑戦する資格を与えられた人を指しています．  本イベントでは，学生やエンジニアなどがチャレンジドと一緒になり，  チャレンジドが社会で活躍するためのアイデア創出を競います．

公共交通オープンデータハッカソン（仮称）

開催日時：2018年12月（予定）
会場：名城大学ドーム前キャンパス・Shake　
主催：アーバンデータチャレンジ愛知拠点 事務局

公共交通データをGoogle Mapsに反映させるだけでは無く，オープンデータ化して様々なサービスやアプリで活用することが重要です．行政，市民，エンジニア，学生が連携してチームを組み，ハッカソンを通じて公共交通の利用促進や街のにぎわい創出などに繋げる新しいアプリやサービスを創り出すことを目指します．

NPO×オープンデータ ワークショップ（仮称）

開催日時：2018年12月（予定）
会場：ウィル愛知　
主催：愛知県
共催：アーバンデータチャレンジ愛知拠点 事務局

愛知県が主催するイベントです．オープンデータ化をするにあたり、NPOなどの市民団体と意見交換を行います．
